
Brain Game

Five (5) short games with randomly generated numbers. In all five games the user has only three (3) attempts to reply correctly. The game is terminated and the user should start a new if/when he/she gives wrong answer.

1. Even or Odd: the user should reply if the number generated is even or odd.

2. Calculator: In this game, the user should solve simple mathematical operation (addition, subtraction, multiplying). Numbers and operation are randomly generated.

3. GCD or the Greatest Common Divisor: The user is presented with two random numbers, and he/she should find the greatest common divisor for two random numbers.

4. Arithmetic progression: The user sees a line of random numbers that create an Arithmetic progression, yet one of the numbers is replaced with two dots ('..'). Length of progression - 10 numbers (not less than 5 numbers). Position of hidden number always different.

5. Prime number: The logic of this game is similar to "Even or Odd". User sees random number and he/she should say whether the number is prime or not.
